# 将缓冲区的内容视为命令,使用特殊解释器来处理

直接选中缓冲区区域，使用`w !bash`，即可将缓冲区的命令在bash中执行。  
1. `w !bash`可以处理下面内容
```
echo 2
```
2. `w !python`可以处理下面内容
```
for i in range(10):
    print(i)
```

如果只有`!`和`解释器`，那么会出现将内容替换为执行后的结果。

**注意`w`与`!`中间的空格，如果不加怎会将当前选中的内容保存为`!`后面的名字的文件**
